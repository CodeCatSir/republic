一、前置使用须知（必看）

家长在使用电子学生证前，请务必熟知以下三条核心规则，避免设备锁卡、失效、无法正常使用：

1、专属SIM卡禁止移植手机使用
电子学生证内置专用物联网卡，严禁拆卸取出插入手机、平板、随身WiFi等其他设备。系统自动监测异常插拔，一经发现将直接注销卡片，导致学生证永久无法通话、定位、考勤，需要重新补卡开通。

2、设备存在省份区域限制，出省需报备
本电子学生证为省内专属设备，绑定固定省份服务权限。学生如需跨省外出、异地就读、假期出省，必须提前联系学校或平台客服报备，完成省份权限变更后才可在外省正常通话和使用功能，未报备出省将出现无信号、无法拨号、定位异常等问题。

3、学生卡号唯一，丢失补办需收费且更换号码
每台学生证对应唯一专属号码与卡片，不支持挂失找回原号码。若设备或卡片丢失，补办新卡需收取40元工本费，邮寄费用另行支付；补办后手机号码将直接变更，家长需要重新绑定设备、添加亲情号码、设置SOS紧急联系人。请妥善保管设备，避免遗失产生额外费用。

二、设备绑定与开通服务

新设备首次使用必须先完成家长端绑定，审核通过后方可缴费、开通全部功能。

操作步骤

  1. 微信打开「护学安」公众号或护学安家长APP，使用学校预留家长手机号登录；
  2. 首页点击【绑定学生设备】，选择电子学生证设备类型；
  3. 扫描机身或包装盒IMEI二维码，或手动输入IMEI编码；
  4. 准确填写学生姓名、年级班级信息，提交绑定申请；
  5. 等待学校后台审核，审核通过后设备状态变为「待开通」，即可进行缴费激活。
注意：一台设备仅可绑定一位主家长账号，IMEI编码请妥善留存，条码磨损可查看设备后盖内部标签。

三、缴费/充值开通套餐

绑定审核通过后，需缴纳服务套餐，才可激活通话、定位、校园考勤等全部功能。

缴费开通流程

  1. 家长端进入【我的设备】，选中对应学生电子学生证；
  2. 点击【充值/续费】，查看月度、年度官方套餐；
  3. 选择对应套餐，核对学生信息与资费金额；
  4. 微信支付完成付款，系统自动激活服务权限；
  5. 缴费成功后重启学生证，5分钟内全部功能生效。
欠费说明：套餐到期未续费，将暂停通话、定位、考勤功能，仅保留SOS紧急呼叫；续费后自动恢复全部功能。

四、亲情号码添加与修改

设备支持设置3组亲情号码，对应机身A1/A2/A3快捷键，学生长按即可一键拨号，仅亲情号可呼入设备。

新增亲情号码步骤

  1. 护学安家长端 → 我的设备 → 亲情号管理；
  2. 点击【新增/编辑】按钮；
  3. 填写联系人称谓与11位大陆手机号；
  4. 分配对应 A1/A2/A3 快捷键，保存即可；
  5. 数据自动同步至设备，无需重启。

修改与删除

在亲情号列表点击编辑可修改号码、称谓，点击删除可清空对应快捷键号码。

限制规则：仅支持中国大陆11位手机号，不支持固话、虚拟小号；陌生号码全部拦截，无法呼入学生设备。

五、SOS紧急联系人设置

SOS紧急呼叫独立于亲情拨号，学生遇到紧急情况长按SOS键即可自动求助，保障学生安全。

  1. 设备详情页进入【安全守护 - SOS设置】；
  2. 可添加1-3名紧急联系人，并设置呼叫顺序；
  3. 系统将按顺序循环拨号两轮，无人接听自动终止；
  4. 紧急呼叫功能不受欠费影响，始终保持可用。

六、设备基础使用方法

1. 拨打电话

长按机身 A1/A2/A3 快捷键一键拨打亲情号码，也可打开设备通讯录手动选择拨号。

2. 接听与挂断

来电按绿色OK键接听，红色挂机键挂断；连续短按两次挂机键可拒接陌生来电。

3. 充电与续航

使用配套USB充电线充电，充满约2小时；正常待机2-3天，低电量自动推送提醒至家长端。

七、常见问题 FAQ

1. 绑定提示IMEI不存在怎么办？+
核对IMEI数字，区分0与字母O；确认设备已录入学校后台；查看机身内部标签重新录入。
2. 缴费后无法打电话？+
重启学生证；确认亲情号已保存同步；检查是否欠费；等待10分钟系统同步生效。
3. 亲情号设置后设备不显示？+
退出家长账号重新登录，删除号码重新保存，重启设备同步数据。
4. 陌生人可以打进电话吗？+
不可以，系统自动拦截所有陌生来电，仅亲情号、SOS联系人可呼入。
5. 可以更换主绑定家长手机号吗？+
设备管理页点击【更换绑定人】，输入新手机号授权即可自动解绑原账号。
