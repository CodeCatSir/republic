一、设备绑定与开通服务

新电子学生证首次使用，需先在护学安家长端完成设备绑定，才能正常缴费、添加亲情号、查看定位。

操作步骤

  1. 微信搜索/打开「护学安」公众号/家长APP，使用登记时预留的家长手机号登录；
  2. 首页点击【绑定学生设备】，选择电子学生证；
  3. 扫描设备背面包装盒/机身IMEI条码，或手动输入IMEI编号；
  4. 填写学生姓名、年级班级，提交绑定；
  5. 学校后台审核通过后，设备状态变为「待开通」，即可进行缴费。
注意：IMEI编号请妥善保管，丢失条码可查看学生证后盖内部标签；一台设备仅支持绑定1个主家长账号。

二、缴费/充值开通套餐

设备绑定审核完成后，需缴纳服务套餐费用，才能激活通话、定位、考勤全功能。

缴费开通流程

  1. 登录护学安家长端，进入【我的设备】，选择对应学生学生证；
  2. 点击【充值/续费】，展示当前可用月度/年度套餐；
  3. 勾选需要开通的套餐，核对学生信息、资费金额；
  4. 微信支付完成付款，系统自动激活服务；
  5. 充值成功后5分钟内，学生证重启即可生效全部功能。
欠费提示：服务到期未续费，学生证仅保留SOS紧急呼叫，普通通话、定位、考勤功能全部暂停；续费后自动恢复。

三、添加/修改亲情号码（学生快捷拨号）

最多可设置3组亲情号码，对应学生证A1/A2/A3快捷键，学生长按按键一键拨号。

新增亲情号步骤

  1. 护学安家长APP/公众号 → 我的设备 → 电子学生证 → 【亲情号管理】；
  2. 点击右上角【编辑/新增】；
  3. 依次填写：称谓（爸爸/妈妈/奶奶等）、11位手机号码；
  4. 分配快捷键A1/A2/A3，点击【保存确认】；
  5. 保存成功，学生证同步更新号码，无需重启设备。

修改/删除亲情号

进入亲情号列表，点击对应联系人右侧编辑按钮，可修改手机号、称谓；点击删除即可清空该快捷键号码。

限制规则：仅支持大陆11位手机号，不支持固话、虚拟运营商副号；陌生号码无法呼入学生证。

四、SOS紧急联系人设置

学生长按设备SOS键，将循环拨打SOS联系人，用于突发安全场景求助，独立于亲情快捷键。

  1. 设备详情页进入【安全守护-SOS设置】；
  2. 支持添加1-3个紧急联系人，设置拨打顺序；
  3. 全部号码无人接听时，循环两轮自动结束呼叫；
  4. 保存后同步至学生证，紧急呼叫不受套餐欠费限制。

五、学生证基础使用方法

1. 拨打电话

长按A1/A2/A3快捷键，直接呼叫对应亲情号；也可打开设备通讯录，选中联系人按OK拨号。

2. 接听与挂断

来电时按绿色OK键接听，红色挂机键挂断；无屏幕操作时，连续短按两次挂机键拒接陌生来电。

3. 充电说明

使用配套USB线充电，充满约2小时；正常待机2-3天，低电量会推送家长提醒。

六、常见问题FAQ

1. 绑定设备提示IMEI不存在怎么办？+
①核对IMEI数字是否输错，区分数字0和字母O；②联系班主任确认设备是否录入学校后台；③检查条码是否磨损，查看机身内部标签。
2. 缴费成功后学生证还是不能打电话？+
①重启电子学生证；②确认亲情号已保存同步；③查看是否欠费未结清；④等待10分钟系统同步生效。
3. 亲情号添加后学生设备不显示？+
退出家长账号重新登录，删除原有亲情号重新保存；将学生证关机再开机同步数据。
4. 陌生号码能打进学生证吗？+
不能，仅已添加的亲情号、SOS联系人可呼入，陌生来电自动拦截，保护学生上课不受打扰。
5. 可以更换绑定的主家长手机号吗？+
在设备管理页点击【更换绑定人】，输入新家长手机号完成授权，原账号自动解绑。
